# dotfiles2 for ubuntu

### git clone
```sh
sudo apt-get install -y git
mkdir -p -v ~/.myconfig
cd ~/.myconfig
git clone https://github.com/camembert-t/dotfiles2.git
```

### setup
```sh
cd ~/.myconfig/dotfiles2
./setup.sh
```
